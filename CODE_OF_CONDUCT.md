# Código de conducta

En interés de fomentar un ambiente abierto y acogedor, quienes contribuyen y mantienen este proyecto se comprometen a que la participación sea una experiencia libre de acoso para todas las personas, sin importar edad, discapacidad, etnia, identidad y expresión de género, nivel de experiencia, nacionalidad, apariencia, raza, religión u orientación.

## Se espera

- Lenguaje acogedor e incluyente.
- Respeto a puntos de vista y experiencias distintas.
- Crítica a las ideas, no a las personas. Aceptar la crítica con apertura.
- Foco en lo que es mejor para el proyecto y el debate público.

## No se acepta

- Lenguaje o imágenes sexualizadas; insultos o ataques personales o políticos.
- Acoso público o privado.
- Publicar información privada de terceros sin permiso.
- Brigading o manipulación coordinada del proceso de contribución.

## Alcance y aplicación

Aplica en todos los espacios del proyecto (issues, pull requests, discusiones). Quien mantiene el repositorio puede editar, mover o cerrar contribuciones que violen este código, y explicará la razón.

Este código adapta, de forma resumida, el Contributor Covenant (https://www.contributor-covenant.org).
